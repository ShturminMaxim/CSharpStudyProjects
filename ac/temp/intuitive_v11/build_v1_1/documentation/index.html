CSS structure

In downloaded package you will find less files, structured in directories:

  • dev-elements - Main elements of layout(navigation, search, rightbar, footer, header)
  • dev-features - All features of this template, such as form-elements, widgets etc.
  • dev-layout - Layout files
  • dev-other - Other files(mixins, function, helpers etc.)
  • dev-plugins - All third-party plugin styles
  • dev-settings - Theme settings(colors,sizes etc.)
  • default.less - Mixin file

In less/css folder of template you will find 6 custom themes for template. Each of this template can be used via including other theme link.

Please use your own css file for your stylesheets. It's easiest way to update template in future.

Javascript structure

Basics

In downloaded package you will find JS folder:

  • plugins - Third-party plugins
  • demo*.js - All demo files that need for theme demonstration
  • dev-app.js - Main application file
  • dev-clock.js - File for lock screen(show clock)
  • dev-email.js - File for email pages
  • dev-faq.js - File for FAQ page
  • dev-icon-search.js - File for list icons
  • dev-layout-default.js - Main application file that controls layout options
  • dev-loaders.js - Loaders
  • dev-settings.js - Settings file(change themes)
  • dev-timeout.js - Session timeout file

How-to...?

How to start using Intuitive template?

  1. Find folder html in downloaded package.
  2. Upload contents of html folder to your webserver
  3. Use blank.html as basic HTML file

How customize template?

Open folder less there you will find all you need to work with template.
In css folder you will find folder dev-settings with .less settings files to customize template styles.
In js folder you will find commented .js files and plugins.

How to change template Theme?

If you using files from html folder:
Open html/css folder. There you will find 10 custom theme files. After you choose one of this themes, change CSS file in your html documents: 

            
    <link rel="stylesheet" type="text/css" id="dev-css" href="css/YourThemeGoesHere.css"/>
